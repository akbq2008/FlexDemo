# FlexDemo

概念部分来自 http://www.ruanyifeng.com/blog/2015/07/flex-grammar.html   
本文只是对概念部分加以demo助于理解。

flex-direction
flex-wrap
flex-flow
justify-content
align-items
align-content

order
flex-grow
flex-shrink
flex-basis
flex
align-self
