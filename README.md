# FlexDemo
这是一个Flex布局的容器属性和项目属性的demo。 This is a demo about  layout of  flex.
概念部分来自 http://www.ruanyifeng.com/blog/2015/07/flex-grammar.html   
本文只是对概念部分加以demo助于理解。
该demo包括了6个容器属性
flex-direction
flex-wrap
flex-flow
justify-content
align-items
align-content
6个项目属性
order
flex-grow
flex-shrink
flex-basis
flex
align-self
